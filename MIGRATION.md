# 📦 插件迁移与安装指南

## 🚀 快速开始（解压即用）

### 1. 下载与安装
```bash
# 1. 下载插件压缩包（包含所有必要文件）
# 2. 解压到 Obsidian 库的插件目录：
你的笔记库/.obsidian/plugins/obsidian-english-assistant/
```

### 2. 文件结构验证
确保以下文件结构完整：
```
obsidian-english-assistant/
├── main.js                 # 主程序文件（必需）
├── manifest.json          # 插件信息文件（必需）
├── styles.css            # 样式文件（必需）
├── resources/
│   └── ecdict.csv        # 77万词词典（必需，约63MB）
├── data.json             # 设置文件（自动生成）
└── 其他文件...
```

### 3. 启用插件
1. 打开 Obsidian
2. 进入 **设置 → 第三方插件**
3. 关闭 **安全模式**
4. 找到 **English Assistant** 并启用
5. 插件会自动初始化所有必要设置

## ✅ 开箱即用功能

### 无需配置即可使用的功能：
- **📖 本地词典查询**：77万词 ECDICT 词典已内置
- **✍️ Shadowing 练习**：支持本地词典标注
- **📊 写作进度追踪**：自动启用
- **📝 语法检查**：基础功能可用
- **💾 知识管理**：自动创建词汇本、语法本

### 需要简单配置的功能：
- **🤖 AI 功能**：需要设置 API Key
  - 进入设置 → English Assistant → AI 配置
  - 选择提供商（OpenAI/DeepSeek/Moonshot等）
  - 输入 API Key

## 🔧 高级迁移场景

### 场景1：迁移到新电脑
1. **完整复制**插件文件夹到新电脑
2. 包含 `data.json` 以保留所有设置和学习记录
3. 重新启用插件即可

### 场景2：迁移到不同的 Obsidian 库
1. 复制整个插件文件夹
2. 如需保留学习数据，复制 `data.json`
3. 如需迁移笔记，复制：
   - `English Vocabulary.md`（词汇本）
   - `Grammar Book.md`（语法本）
   - `Mistake Book.md`（错题本）

### 场景3：团队共享
1. 分享插件文件夹（包含词典）
2. 每个成员自行配置 API Key
3. 学习数据独立存储，不会冲突

## 🆘 常见问题

### Q: 词典加载失败？
A: 检查 `resources/ecdict.csv` 文件是否存在且完整（约63MB）

### Q: 设置丢失？
A: 插件会自动使用默认设置，也可从 `data.json` 恢复

### Q: API 功能不可用？
A: 进入设置配置 API Key，本地词典功能不受影响

### Q: 文件保存失败？
A: 插件会自动创建必要文件夹，确保有写入权限
