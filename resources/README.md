# Local Dictionary

This plugin does not bundle `ecdict.csv` in the GitHub repository or release package.

To enable offline dictionary lookup:

1. Download `ecdict.csv` or `ecdict.mini.csv` from the official ECDICT project:
   - <https://github.com/skywind3000/ECDICT>
   - <https://github.com/skywind3000/ECDICT/releases>
2. Rename the file to `ecdict.csv` if needed.
3. Place it in this folder:

```text
.obsidian/plugins/obsidian-english-assistant/resources/ecdict.csv
```

The plugin also supports other ECDICT-compatible CSV dictionaries.
